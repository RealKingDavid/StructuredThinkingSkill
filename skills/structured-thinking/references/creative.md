# Creative Project Framework

## Phases

### Phase 1: Concept & Vision
Lock in the creative direction before producing anything.
- Tasks: Define the core idea, identify audience and tone, research references, set scope
- Subtasks: Write a one-paragraph pitch or logline, gather 5–10 reference works, define what makes this unique, identify format/medium/length, clarify what success looks like

### Phase 2: Structure & Plan
Map out the full shape of the project before diving into creation.
- Tasks: Create an outline or structure, plan production timeline, identify dependencies and assets needed
- Subtasks: Outline acts/chapters/episodes/sections, identify what needs to be created vs. sourced, list collaborators or tools needed, define milestones, build a realistic schedule with buffer

### Phase 3: Create (First Draft / First Pass)
Produce the raw material — quantity over perfection here.
- Tasks: Generate first draft or rough cut, capture all ideas without filtering, maintain creative momentum
- Subtasks: Set daily/weekly creation targets, use timed creative sessions (e.g., Pomodoro), capture everything even if rough, note areas to revisit, resist editing while creating

### Phase 4: Revise & Refine
Improve the work through structured editing and feedback.
- Tasks: Self-edit against original vision, get feedback from trusted sources, revise systematically
- Subtasks: Read/view the full work with fresh eyes, list what's working vs. not, gather 2–3 external readers/reviewers, identify structural issues before line-level fixes, do multiple passes (macro → micro)

### Phase 5: Finalize & Share
Polish, produce, and get the work out into the world.
- Tasks: Final production/formatting, choose distribution channels, launch and promote
- Subtasks: Final proofread or technical pass, prepare all assets (cover, metadata, credits), choose platform/venue/channel, create a launch or release plan, define how you'll gather audience feedback

## Key Risks to Surface
- Perfectionism stalling the creation phase — first drafts are supposed to be rough
- Scope creep — the project expanding beyond what's achievable
- Creating in isolation without any external feedback
- Skipping structure and losing the thread mid-project
- Not planning for the "last 20%" — production, publishing, and promotion take real time

## Type-specific notes
- Writing: Separate drafting from editing — they use different cognitive modes
- Film/video: Pre-production investment saves enormous time in production
- Games: Prototype the core mechanic first; don't build content around a broken loop
- Music: Finish songs before perfecting them — production polish is a trap early on
- Art: Define the series or scope before starting individual pieces
